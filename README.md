# MunuSync - AI-Powered Recruitment Platform

MunuSync is a comprehensive, AI-driven recruitment ecosystem that connects job seekers with opportunities through intelligent matching, streamlined workflows, and seamless user experiences across mobile and web platforms. Built as a **multi-tenant system**, every company gets their own space, every job seeker has their personalized journey, and everything is connected under one scalable infrastructure.
